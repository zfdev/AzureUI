Azure UI Webfont IconsLast Update 201511251614

What it is and why we use it.

Azure UI Webfont Icons gives you scalable vector icons that can instantly be customized, such as size, color, drop shadow, and anything that can be done with the power of CSS. In addition, it displays gorgeously on high-resolution screen.

How to use it?

Step 1:

Download the zip file form here. Put all the folder under the root directory.

Step 2:

Include the style sheet file:

Example:

icon-xiaoice
